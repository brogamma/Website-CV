Projects

On this page the projects i have done in my time at school
are displayed. These project are all done in collaboration with other students.

Smart fuse Project

The smart fuse project is an project that had the assignment to make an module that measures the voltage, current and other factors from the power grid. This project is done at Alliander N.V. in the Netherlands.

Amorace

This project is done for the study association Amoras of engineering faculty at the HAN. This project was to make an event that let people race RC cars with VR goggles. This was achieved by mounting a camera on the RC cars and syncing them up with the VR goggles.

Incubator Project

This project was about making a Incubotor togheter with mechanical enginering students and two other embedded systems enginering students. For this project i made a website with php elements to display data form a database. The data in this database came form the incubator we built with some microcontrollers.
